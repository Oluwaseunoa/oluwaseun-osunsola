Background

I'm a Cloud & DevOps Engineer and Cybersecurity Practitioner with proven, hands-on experience building and securing cloud-native infrastructure across AWS and Azure.

I design and implement end-to-end CI/CD pipelines using GitHub Actions and Jenkins, containerize applications with Docker and Kubernetes, and automate infrastructure provisioning with Shell Scripting and Terraform — reducing deployment friction and increasing delivery reliability.

I am actively seeking roles in DevOps Engineering, Cloud Engineering, Cloud Security, or Site Reliability Engineering (SRE) where I can contribute immediately and grow with a forward-thinking team.
